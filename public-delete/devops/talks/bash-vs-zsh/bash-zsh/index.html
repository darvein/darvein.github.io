Global qualifiers 

ls **/*.(dll|py)

# list the 3rd file
ls *(.[3])

# match files only, no dirs, no special files
du -h *(.)

# match executable files
du -h *(*.)

# files by size
file *(Lm+1)
ls *(.L0)

# latest two files
file git*(om[1,2])

# files by modified date
vim *(m0)
ls *(.m+2)
ls *(.^m0)

# files by permissions
ls *(.f644)

# files by permissions
ls *(.f644)
ls *(.u:apache:)

History in zsh 

!!    # repeat last
!-1   # repeat previous of last
!!0   # take previous command
!^    # take first arg from last
!*    # take all params from last

VI Mode

Bash: set -o vi

Zsh: bindkey -v

ZSH builtin functions

Directories history:

  • cd -<TAB>
  • pushd
    • cp file ~1

ZMV:

zmv '(*).jpeg' '$1.jpg'

Long directory shortcut with hash:

hash -d sh=$PWD

Subprocess in bash and zsh

With new process:

$ (id)

$ cat <(date)

$ echo $date

$ ls 1> >(grep demo >output.log)

Without new process:

$ {cd ~; ls}

Arrays 

print $array[2]

echo "${arr[2]}"
